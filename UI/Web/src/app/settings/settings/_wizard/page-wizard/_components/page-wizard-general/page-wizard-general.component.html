Pages form the central way to search and download content with Media-Provider, each page will be a tab in the navigation bar. And allow you to search content from at least one provider. You'll be able to adjust on which things you can filter while downloading. For more information see the wiki.

A page must have a title, at least one provider, and may have an icon. The list of names for the icons can be found here
